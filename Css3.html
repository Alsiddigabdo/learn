CSS3: تعلم أحدث تحديثات ورق الأنماط للويب

في CSS3 تم إضافة العديد من المحددات (selectors) الجديدة التي تساعد في تحديد العناصر بشكل أكثر دقة. بعض الأمثلة على المحددات الجديدة هي nth-child و nth-of-type والتي تستخدم لاختيار عنصر معين في نسل العنصر الأصلي.


ul li:nth-child(odd) {
  background-color: lightgray;
}

هذا الكود يقوم باختيار كل عنصر فردي في عناصر القائمة (li) داخل العنصر (ul) ويعطيه خلفية رمادية فاتحة.

الخاصية Box Model (نموذج المربع):

هذه الخاصية تُمكنك من تحديد حجم وتنسيق العناصر. تحتوي على عناصر مثل width و height و margin و padding.


div {
  width: 300px;
  height: 200px;
  margin: 20px;
  padding: 10px;
  background-color: lightblue;
}

هذا الكود يحدد عرض وارتفاع العنصر (div) إلى 300 بكسل و 200 بكسل على التوالي ويضيف هوامش قيمتها 20 بكسل وحواف داخلية قيمتها 10 بكسل، ويعطي العنصر خلفية لونها أزرق فاتح.

الخاصية Transitions (الانتقالات):

تسمح لك خاصية Transitions بتعريف تأثيرات الانتقال السلسة بين مراحل مختلفة لعناصر HTML.


button {
background-color: blue;
color: white;
transition: background-color 0.5s ease;
}
button:hover {
background-color: red;
}

هذا الكود يحدد أن لون خلفية الزر سيتغير من الأزرق إلى الأحمر عند تحويل المؤشر عليه بسرعة 0.5 ثانية.

الخاصية Flexbox (صندوق المرونة):

تتيح لك ترتيب وتنسيق العناصر داخل العنصر الأب بشكل مرن ومناسب لعرض الصفحة على مختلف الأجهزة.


.container {
  display: flex;
  justify-content: space-between;
}

هذا الكود يضع العناصر داخل العنصر (container) بترتيب مرن، مع توزيع المساحة بين العناصر بشكل متساوٍ.

الخاصية Animations (التحريكات):

تسمح لك بإنشاء تحريكات للعناصر باستخدام سلاسة وتوقيت محدد.


@keyframes slide {
  0% {
    transform: translateY(0);
  }
  100% {
    transform: translateY(100px);
  }
}

.element {
  animation-name: slide;
  animation-duration: 2s;
  animation-iteration-count: infinite;
}

هذا الكود ينشئ تحريكًا للعنصر (element)، حيث يتحرك باتجاه الأسفل بمقدار 100 بكسل، وتستمر مدة الحركة لمدة 2 ثانية بشكل مستمر.

الخاصية Media Queries (استعلامات الوسائط):

تتيح لك ضبط تنسيقات العناصر بناءً على خصائص الشاشة المختلفة مثل العرض والارتفاع.


@media (max-width: 600px) {
  body {
    background-color: lightgray;
  }
}

هذا الكود يقوم بتغيير لون خلفية الصفحة إلى رمادي فاتح عندما يكون عرض الشاشة أصغر من 600 بكسل.

الخاصية Transitions (الانتقالات):

تُسمح لك بإضافة تأثيرات انتقال سلسة للعناصر عند حصول تغيير في خصائصها مثل اللون والحجم والشفافية.


.element {
  transition-property: background-color;
  transition-duration: 1s;
}

.element:hover {
  background-color: blue;
}

هذا الكود يتيح تحوّل لون العنصر (element) إلى اللون الأزرق بشكل سلس عند تحويل المؤشر فوقه.

الخاصية Grid (الشبكة):

تتيح لك إنشاء تخطيطات مرنة ومتعددة الأعمدة والصفوف بسهولة وفاعلية.


.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: gray;
}

هذا الكود ينشئ تخطيطًا شبكيًا مكونًا من صفين، حيث تكون العناصر (item) بلون رمادي وتفصل بمسافة 20 بكسل بينها.

الخاصية Variables (المتغيرات):

تُسمح لك بتعريف متغيرات واستخدامها في مختلف الخصائص الأخرى، مما يُسهِّل إعادة استخدام القيم وتحديثها بسهولة.


:root {
  --main-color: blue;
  --secondary-color: yellow;
}

.element {
  background-color: var(--main-color);
  color: var(--secondary-color);
}

هذا الكود يعرِّف المتغيرات --main-color و--secondary-color ويُستخدم في العنصر (element) لتطبيق لونين مختلفين في الخلفية والنص.

الخاصية Box Shadow (الظلال):

تُمكنك من إضافة ظلال للعناصر لإضفاء أبعاد ثلاثية عليها وتحسين مظهرها.


.element {
  box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2);
}

هذا الكود يُنشئ ظلالًا للعنصر (element) بعرض 2 بكسل وارتفاع 2 بكسل وعمق 4 بكسل بلون أسود ذو انعكاسية 0.2.

الخاصية Flexbox (المرونة):

تُمكِّنك من إنشاء تخطيطات مرنة وسهلة التنظيم للعناصر داخل العناصر الأساسية.


.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
}

.item {
  flex: 1;
}

هذا الكود يُنشئ تخطيطًا مرنًا للعناصر باتجاه أفقي، حيث تكون المسافة بينها متساوية وتمتد عبر المساحة المتاحة بالتساوي.

الخاصية Animations (الحركات):

تُسمح لك بإضافة حركات وتأثيرات متحركة للعناصر، مثل التحرك والتدوير والتحجيم.


.element {
  animation-name: slide-in;
  animation-duration: 2s;
}



@keyframes slide-in {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

هذا الكود يُنشئ حركة للعنصر (element) يتحرك في الأفق ببداية من اليسار (-100%) إلى الوسط (0) خلال 2 ثانية.

13. الخاصية Transform:

تستخدم لتعديل الشكل والموقع والحجم والزاوية للعناصر في صفحة الويب.


.element {
  transform: rotate(45deg);
}

هذا الكود يدوّر العنصر (element) بزاوية 45 درجة.

14. الخاصية Transition:

تستخدم لإضافة تأثيرات انتقال سلسة للعناصر عند تغيير حالتها أو عند مرور المؤشر عليها.


.element {
  transition: background-color 0.3s ease-in-out;
}

.element:hover {
  background-color: blue;
}

هذا الكود يُنشئ تأثيرًا انتقاليًا للعنصر (element) عند تغيير لون الخلفية، حيث يستغرق الانتقال 0.3 ثانية بتصفيف سهل الانتقال.

15. الخاصية Grid:

تتيح لك إنشاء تخطيطات شبكية مُرنة لتنظيم العناصر ثنائية الأبعاد (مثل الجداول) في صفحة الويب.


.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-gap: 20px;
}

هذا الكود يُنشئ تخطيطًا شبكيًا للعناصر في الحاوية (container) مع عمودين متساويين وفراغ بينهما بعرض 20 بكسل.
