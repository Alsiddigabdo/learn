مصطلحات في JavaScript

  • Console.log(typeof المدخلات); - يقوم بطباعة نوع القيمة المدخلة في وحدة التحكم.
  • Boolean - يستخدم لتمثيل القيم المنطقية (صح أو خطأ).
  • Object - يُستخدم لتمثيل هيكل معين أو كائن في اللغة.
  • Null - يُستخدم لتعيين قيمة فارغة أو غير معرفة لمتغير.
  • String - يُستخدم لتمثيل سلاسل النصوص.
  • Undefined - يُستخدم لتعيين متغير غير معرف ولم يتم تعيين أي قيمة له بعد.

عوامل المقارنة (Comparison Operators)

تستخدم للمقارنة بين القيم وإرجاع قيمة منطقية (صح أو خطأ).

  • == - يقارن إذا كانت القيم متساوية دون الاهتمام بنوع البيانات.
  • === - يقارن إذا كانت القيم متساوية مع الاهتمام بنوع البيانات.
  • != - يقارن إذا كانت القيم غير متساوية.
  • < - يقارن إذا كانت القيمة الأولى أصغر من القيمة الثانية.
  • > - يقارن إذا كانت القيمة الأولى أكبر من القيمة الثانية.
  • <= - يقارن إذا كانت القيمة الأولى أصغر من القيمة الثانية أو تساويها.
  • >= - يقارن إذا كانت القيمة الأولى أكبر من القيمة الثانية أو تساويها.

العوامل المنطقية (Logical Operators)

  • ! - يقوم بعملية النفي (العكس) للقيمة المنطقية.
  • && - يقوم بالمقارنة الشرطية "و" (AND).
  • || - يقوم بالمقارنة الشرطية "أو" (OR).

تتيح دالة "join" في لغة الجافا اسكريبت دمج عناصر مصفوفة في سلسلة واحدة، حيث تأخذ المصفوفة كمدخل وتقوم بإرجاع سلسلة متكونة من عناصر المصفوفة المفصولة بفاصلة (أو أي حرف آخر يتم تحديده). الصيغة العامة لدالة "join" هي: array.join(separator) حيث: array: المصفوفة التي سيتم دمج عناصرها في سلسلة. separator: الفاصلة (أو أي حرف آخر يتم تحديده) الذي سيتم استخدامه لفصل العناصر المختلفة في السلسلة المرجعة.

Loop - For


/*
    Loop
        - For
            for ([1] [2] [3]) {
                Block Of Code
            }
*/

for (let i = 0; i < 10; i++) {
    console.log(i);
}

هذا الكود هو مثال بسيط لاستخدام حلقة التكرار "for" في لغة الجافا سكريبت. حلقة التكرار "for" هي تقنية شائعة في البرمجة تستخدم لتكرار تنفيذ سلسلة من الأوامر عدة مرات، وتسمح لنا بتحديد عدد المرات التي نريد تكرار الأوامر بها. تتكون حلقة "for" من ثلاثة أجزاء: [1] متغير يستخدم لتحديد قيمة بداية الحلقة (يتم تعيين قيمته قبل بدء التكرار). [2] تعبير مشروط يحدد متى يتوقف التكرار (يتم تقييمه في كل مرة يتم فيها تنفيذ الحلقة). [3] تعبير يتم تنفيذه في كل مرة يتم فيها تنفيذ الحلقة. في الكود المعطى، تم استخدام حلقة "for" لتنفيذ الأمر console.log(i) عشر مرات، حيث تبدأ الحلقة من القيمة 0 وتستمر حتى تصل القيمة إلى 9 (أي أقل من 10)، وفي كل مرة يتم فيها تنفيذ الحلقة، يتم زيادة قيمة المتغير i بمقدار 1 باستخدام العبارة i++ التي تعني "زيادة i بمقدار 1". يتم طباعة القيم التي تم تخزينها في المتغير i في كل مرة يتم فيها تنفيذ الحلقة، باستخدام دالة console.log() التي تقوم بعرض النتيجة في وحدة التحكم. وبالتالي، يتم طباعة القيم من 0 إلى 9.

        - Loop On Sequences
*/
let myFriends = [1, 2, "Osama", "Ahmed", 3, 4, "Sayed", 6, "Ali"];
let onlyNames = [];

for (let i = 0; i < myFriends.length; i++) {
if (typeof myFriends[i] === "string") {

}
}

هذا الكود يستخدم حلقة "for" للتكرار على جملة مصفوفة "myFriends" والبحث عن العناصر التي تحتوي على النصوص فقط. وفي كل مرة يتم فيها تنفيذ الحلقة، يتم التحقق مما إذا كان العنصر الحالي من النوع "string"، وإذا كان كذلك، يتم إضافة العنصر إلى مصفوفة جديدة تسمى "onlyNames" باستخدام الأمر "onlyNames.push(myFriends[i])". بمجرد الانتهاء من تنفيذ الحلقة، يتم عرض نتيجة الأسماء فقط باستخدام دالة "console.log()"، ويتم طباعة قيم المصفوفة "onlyNames" الذي يحتوي على العناصر التي تحتوي على النصوص فقط.

Return Nested Function 

        

تستخدم الدوال المتداخلة في الجافا سكريبت لإرجاع دالة من داخل دالة أخرى، ويمكن استخدام هذا النوع من الدوال للحفاظ على سرية بعض المتغيرات أو الدوال في الدالة الرئيسية.

هنا مثال بسيط للدوال المتداخلة:

function outerFunction() {
    let outerVariable = "Hello, ";

    function innerFunction(name) {
        return outerVariable + name;
    }

    return innerFunction;
}

let greeting = outerFunction();
console.log(greeting("John")); // Output: Hello, John

Arrow Function

        
Arrow Function هي طريقة جديدة لتعريف واستخدام الدوال في JavaScript، والتي تم تقديمها في ES6.
ويمكن الاستفادة منها إذا كانت ون استيك اي سطر واحد

تميز Arrow Function عن الدوال التقليدية في الشكل البسيط والمختصر لتعريفها، حيث تستخدم الرمز "=>" للإشارة إلى الدالة، ولا يحتاج تعريفها إلى استخدام الكلمة الرئيسية "function".

فيما يلي مثال عملي على استخدام Arrow Function:

// تعريف دالة تقليدية باستخدام الكلمة الرئيسية function
function multiply(a, b) {
    return a * b;
}

// تعريف الدالة نفسها باستخدام Arrow Function
const multiplyArrow = (a, b) => a * b;

// استخدام الدالتين
console.log(multiply(2, 3)); // سيظهر الناتج 6
console.log(multiplyArrow(2, 3)); // سيظهر الناتج 6

Global And Local Scope

        


في جافا سكريبت، الـ Scope هو المجال الذي يمكن للمتغيرات والدوال أن تكون متاحة فيه، وينقسم الـ Scope إلى نطاقين: Global Scope و Local Scope.

Global Scope:
هو المجال الذي تعرف فيه المتغيرات والدوال خارج الدوال والكائنات، وبالتالي يمكن الوصول إليها من أي مكان في الكود.

Local Scope:
هو المجال الذي تعرف فيه المتغيرات والدوال داخل الدوال أو الكائنات، وبالتالي لا يمكن الوصول إليها من خارج هذه الدوال أو الكائنات.

في الشفرة التالية يوضح مثال بسيط لفهم Global و Local Scope:

// تعريف المتغير العالمي
let globalVariable = 5;

function myFunction() {
    // تعريف المتغير المحلي
    let localVariable = 10;

    console.log("globalVariable في myFunction: ", globalVariable); // سيظهر الناتج 5
    console.log("localVariable في myFunction: ", localVariable); // سيظهر الناتج 10
}

myFunction();

console.log("globalVariable في النطاق العالمي: ", globalVariable); // سيظهر الناتج 5
console.log("localVariable في النطاق العالمي: ", localVariable); // سيظهر خطأ Uncaught ReferenceError

Scope - Block Scope [If, Switch, For]

        


في جافا سكريبت، الـ Scope هو المجال الذي يمكن للمتغيرات والدوال أن تكون متاحة فيه، وينقسم الـ Scope إلى نطاقين: Global Scope و Local Scope.

بالإضافة إلى الـ Global Scope و Local Scope، هناك مفهوم آخر في الـ Scope يسمى Block Scope. يشير Block Scope إلى المتغيرات التي يتم تعريفها داخل كتلة الشفرة، مثل If Statement و For Loop و Switch Statement.

لنفترض أن لدينا مثالاً بسيطاً يستخدم If Statement لفحص قيمة المتغير، ويحتوي على متغير يتم تعريفه داخل If Statement، سيكون الكود كالتالي:

if (true) {
  let myVariable = "Hello";
  console.log(myVariable); // سيظهر الناتج "Hello"
}

console.log(myVariable); // سيظهر خطأ Uncaught ReferenceError

Lexical Scope 

        

في جافا اسكريبت، الـ Scope هو المجال الذي يمكن للمتغيرات والدوال أن تكون متاحة فيه. وينقسم الـ Scope إلى نطاقين: Global Scope و Local Scope.

أما عن Lexical Scope، فهو مفهوم يتعلق بالـ Scope الخاص بالدوال. ويشير Lexical Scope إلى الـ Scope الذي تم تعريف الدالة فيه، حيث يمكن للدالة الوصول إلى المتغيرات التي تم تعريفها في نفس النطاق الذي تم تعريف الدالة فيه.

لنفترض أن لدينا مثالًا بسيطًا:

let a = "Hello";

function myFunction() {
  console.log(a);
}

myFunction(); // سيظهر الناتج "Hello"

function بعض خصائص ال

        
خصائص الـ function في الجافا اسكريبت:

1- يمكن تعريف الـ function بشكل عام كالتالي:

function functionName(parameter1, parameter2, ...) {
    // code to be executed
}

حيث يمكن إضافة معاملات (parameters) للـ function واستخدامها داخل الـ function.

مثال:

function greet(name) {
    console.log("Hello " + name);
}

greet("John"); // Hello John

2- يمكن إرجاع قيمة من الـ function باستخدام return statement.

مثال:

function add(a, b) {
    return a + b;
}

console.log(add(2, 3)); // 5

3- يمكن تعريف الـ function ك expression وتخزينها في متغير.

مثال:

var greet = function(name) {
    console.log("Hello " + name);
}

greet("John"); // Hello John

4- يمكن استخدام IIFE (Immediately Invoked Function Expression) لتنفيذ الـ function فور تعريفها.

مثال:

(function() {
    console.log("I am an IIFE");
})();
// Output: I am an IIFE

5- يمكن استخدام callback functions لتنفيذ الـ function بعد انتهاء عملية أخرى.

مثال:

function greet(name, callback) {
    console.log("Hello " + name);
    callback();
}

function sayGoodbye() {
    console.log("Goodbye!");
}

greet("John", sayGoodbye);
// Output: Hello John
//         Goodbye!

6- يمكن تعريف الـ function ك arrow function.

مثال:

const add = (a, b) => a + b;

console.log(add(2, 3)); // 5

7- يمكن استخدام default parameters لتحديد قيمة افتراضية للمعاملات.

مثال:

function greet(name = "World") {
    console.log("Hello " + name);
}

greet(); // Hello World
greet("John"); // Hello John

Higher Order Functions map

        


درس Higher Order Functions في جافا سكريبت يتعلق بالقدرة على استخدام الدوال كمتغيرات وتمريرها كوسائط لدوال أخرى. واحدة من الدوال العليا هي دالة "map" التي تستخدم لتحويل عناصر مصفوفة إلى عناصر جديدة بناءً على دالة تطبق على كل عنصر.

لنفترض أن لدينا مصفوفة تحتوي على أعداد ونريد ضرب كل عنصر فيها في 2. بدلاً من استخدام حلقة لتحقيق ذلك، يمكننا استخدام دالة "map" لتنفيذ العملية بسهولة.

فيما يلي مثال بسيط لاستخدام دالة "map" في جافا سكريبت:

// تعريف المصفوفة
var numbers = [1, 2, 3, 4, 5];

// تعريف الدالة التي ستطبق على كل عنصر
function multiplyByTwo(number) {
    return number * 2;
}

// استخدام دالة map لتحويل الأعداد
var multipliedNumbers = numbers.map(multiplyByTwo);

// طباعة النتائج
console.log(multipliedNumbers);

في هذا المثال، قمنا بتعريف مصفوفة "numbers" تحتوي على الأعداد [1, 2, 3, 4, 5]. ثم قمنا بتعريف الدالة "multiplyByTwo" التي تأخذ عددًا وترجع ناتج ضربه في 2.

ثم استخدمنا دالة "map" على المصفوفة "numbers" ومررنا إليها الدالة "multiplyByTwo". تقوم دالة "map" بتطبيق الدالة المعطاة على كل عنصر في المصفوفة وتخزين النتائج في مصفوفة جديدة تسمى "multipliedNumbers".

أخيرًا، قمنا بطباعة مصفوفة "multipliedNumbers" والتي تحتوي على الأعداد المضاعفة.

النتيجة المطبوعة ستكون: [2, 4, 6, 8, 10]، حيث تم ضرب كل عنصر في المصفوفة الأصلية.

Higher Order Functions – Filter 

        


درس Higher Order Functions - Filter يعتبر من الدروس المهمة في لغة جافا سكريبت. تسمح لك هذه الوظيفة بتطبيق وظيفة معينة على مجموعة من العناصر وفقًا لشرط محدد، واسترجاع مجموعة جديدة تحتوي على العناصر التي تم تطبيق الشرط عليها.

لفهم هذا الدرس بشكل أفضل، دعنا نأخذ مثالًا بسيطًا: لدينا مصفوفة تحتوي على أرقام، ونحن بحاجة إلى استرجاع جميع الأرقام التي تكون أكبر من 5. سنستخدم وظيفة filter لتنفيذ ذلك.

في البداية، يمكننا إنشاء مصفوفة تحتوي على الأرقام المطلوبة. على سبيل المثال، لنفترض أن لدينا المصفوفة التالية:

const numbers = [1, 2, 6, 8, 3, 7, 9, 4];

الآن، سنقوم بتعريف وظيفة تسمى الشرط أو predicate. في حالتنا، سيكون الشرط هو أن العدد يجب أن يكون أكبر من 5. سنقوم بتعريف الشرط على النحو التالي:

const condition = (number) => number > 5;

الآن، بعد تحديد المصفوفة والشرط، سنستخدم وظيفة filter لتنفيذ الشرط على كل عنصر في المصفوفة واسترجاع المصفوفة الناتجة:

const filteredNumbers = numbers.filter(condition);
console.log(filteredNumbers);

إذا قمت بتشغيل هذا الكود، ستحصل على النتيجة التالية في وحدة التحكم:

[6, 8, 7, 9]

المصفوفة الناتجة تحتوي على الأرقام 6، 8، 7، 9 فقط، وهي الأرقام التي تم تطبيق الشرط (أن تكون أكبر من 5) عليها.

هكذا، تستخدم وظيفة filter لتطبيق شرط معين على مجموعة.

Higher Order Functions - Reduce

        


درس Higher Order Functions - Reduce هو درس مهم في لغة جافا سكريبت، حيث يتيح لك هذا الدرس تجميع عناصر مصفوفة إلى قيمة واحدة واحدة. يستخدم الدالة reduce لتنفيذ هذه العملية.

لنفهم الدرس بشكل أفضل، دعنا نستخدم مثالًا بسيطًا. لنفترض أن لدينا مصفوفة تحتوي على أرقام ونريد حساب المجموع الإجمالي لجميع الأرقام في المصفوفة.

لنبدأ بإنشاء المصفوفة. لنفترض أن لدينا المصفوفة التالية:

const numbers = [1, 2, 3, 4, 5];

الآن، سنستخدم وظيفة reduce لتجميع الأرقام في المصفوفة واسترجاع المجموع الإجمالي. سنستخدم دالة الاسترجاع (callback function) لتحديد العملية التي سيتم تطبيقها على الأرقام. في هذه الحالة، سنستخدم عملية الجمع.

const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(sum);

تم استخدام القيمة الأولية 0 كمعامل ثاني لدالة reduce. القيمة الأولية هو القيمة التي ستتمتع بها المتغير المجمع (accumulator) في البداية. في هذه الحالة، سنبدأ بالجمع من القيمة 0.

إذا قمت بتشغيل هذا الكود، ستحصل على النتيجة التالية في وحدة التحكم:

15

الناتج 15 هو المجموع الإجمالي لجميع الأرقام في المصفوفة.

هذا هو مثال بسيط للدرس reduce في جافا سكريبت. يمكنك استخدام وظيفة reduce لتنفيذ العمليات الأخرى مثل الضرب أو إيجاد القيمة القصوى أو أي عملية أخرى تتطلب تجميع العناصر في مصفوفة إلى قيمة واحدة واحدة.

forEach جافا اسكريبت

        



 وظيفة تُستخدم لتنفيذ دالة معينة لكل عنصر في مصفوفة أو قائمة.

هذا هو تعريف بسيط لاستخدام forEach:

array.forEach(function(currentValue, index, array) {
  // قم بتنفيذ العمليات المطلوبة على currentValue

توضح البارامترات التالية ما تمثله:

currentValue: قيمة العنصر الحالي في المصفوفة أو القائمة.
index: الفهرس (الموضع) الحالي للعنصر.
array: المصفوفة أو القائمة التي تم تطبيق forEach عليها.

الآن دعنا نرى مثالًا بسيطًا لكيفية استخدام forEach:

let numbers = [1, 2, 3, 4, 5];

numbers.forEach(function(number) {
  console.log(number * 2);
});

في هذا المثال، يتم استخدام forEach لتكرار المصفوفة "numbers" وضرب كل عنصر فيها بمقدار 2. ستتم طباعة النتائج التالية:

2
4
6
8
10

يتم تنفيذ الدالة المرفقة بـ forEach لكل عنصر في المصفوفة وتمرير القيمة الحالية كمعامل للدالة. في هذا المثال، يتم ضرب كل عنصر في المصفوفة بمقدار 2 وتم طباعة النتائج.

تمثل forEach طريقة سهلة ومرنة لتنفيذ إجراء على كل عنصر في مصفوفة أو قائمة بدون الحاجة إلى حلقات تكرار محددة.

دالة getElementById في جافا سكريبت

في جافا سكريبت، يُستخدم الدالة getElementById للحصول على عنصر واجهة المستخدم في صفحة HTML باستخدام معرف (ID) العنصر.

تعريف دالة getElementById

        
document.getElementById(elementId);

تأخذ الدالة getElementById واحدًا من البارامترات التالية:

  • elementId: معرف العنصر (ID) الذي ترغب في الحصول عليه.

مثال بسيط

لنفترض أن لدينا صفحة HTML تحتوي على العناصر التالية: 

        
<h1 id="myHeading">مرحبا بك في صفحة الويب</h1>
<p id="myParagraph">هذا هو فقرة نص توضيحي.</p>

يمكننا الآن استخدام دالة getElementById للحصول على قيمة عنصر العنوان والفقرة، وذلك باستخدام معرفاتهما: 

        
let headingElement = document.getElementById("myHeading");
let paragraphElement = document.getElementById("myParagraph");

console.log(headingElement.textContent); // سيتم طباعة نص عنصر العنوان
console.log(paragraphElement.textContent); // سيتم طباعة نص عنصر الفقرة

في هذا المثال، يتم استخدام getElementById للحصول على العنصر المرتبط بمعرف "myHeading" وتخزينه في المتغير headingElement. ثم يتم استخدام getElementById مرة أخرى للحصول على العنصر المرتبط بمعرف "myParagraph" وتخزينه في المتغير paragraphElement. ثم يتم طباعة نص كل عنصر باستخدام خاصية textContent.

تُستخدم getElementById عادةً في جافا سكريبت للتعامل مع عناصر واجهة المستخدم وتعديلها أو الحصول على القيم المدخلة من المستخدم. يُعتبر العنصر الذي يتم استرجاعه من getElementById ككائن DOM ويمكن التلاعب به بحرية باستخدام خصائص وطرق أخرى من DOM.

DOMال

DOM (Document Object Model): DOM هو نموذج يُمثل هيكلة ومحتوى صفحة الويب ككائنات. يتيح لك الوصول إلى العناصر على الصفحة وتحريرها وتعديلها باستخدام JavaScript. يمكنك التفاعل مع العناصر النصية والصور والروابط والأحداث وغيرها الكثير. فيما يلي مثال بسيط يوضح كيفية استخدام DOM للوصول إلى عنصر في الصفحة وتعديله:

// استهداف العنصر باستخدام الـ id
var heading = document.getElementById("myHeading");
// تغيير نص العنصر
heading.innerHTML = "مرحباً بك في العالم!";
// استهداف الزر باستخدام الـ id
var button = document.getElementById("myButton");
// إضافة حدث النقر لتغيير العنوان
button.addEventListener("click", function() { heading.innerHTML = "لقد قمت بالنقر!"; });

forEach: forEach هو أسلوب في JavaScript يُستخدم لتنفيذ وظيفة مُعينة لكل عنصر في مصفوفة أو قائمة. يعتبر forEach أحد أساليب العناصر الأكثر استخدامًا في برمجة JavaScript. فيما يلي مثال يوضح كيفية استخدام forEach لتكرار عناصر المصفوفة وطباعتها:

var numbers = [1, 2, 3, 4, 5];
numbers.forEach(function(number) { console.log(number); });

Browser Object Model (BOM)
 

            
هو نموذج يقدمه المتصفح لتمثيل المستند المحمل ونوافذ المتصفح والعناصر المرتبطة بها ككائنات قابلة للتلاعب.
 تعتبر واجهة BOM غير موحدة بين المتصفحات وتختلف في بعض النقاط، ولكن سوف أقدم لك العناصر الأكثر استخدامًا في BOM مع بعض الأمثلة.

        
1. Window Object:
        الكائن Window يمثل نافذة المستعرض. يحتوي على العديد من الخصائص والطرق المفيدة مثل:
        - alert(): عرض رسالة تنبيه للمستخدم.
        - confirm(): عرض رسالة تأكيد للمستخدم.
        - prompt(): عرض رسالة تطلب إدخال البيانات من المستخدم.

        مثال:
        

          alert("مرحبًا!");
          let result = confirm("هل أنت متأكد؟");
          let name = prompt("ما اسمك؟");
        


        2. Document Object:
        الكائن Document يمثل المستند المحمل في النافذة الحالية ويسمح بتلاعب العناصر الموجودة داخله. قد ترغب في استخدام العديد من خصائص وطرق الكائن Document مثل:
        - getElementById(): استرجاع عنصر بناءً على الهوية.
        - getElementsByTagName(): استرجاع مجموعة من العناصر استنادًا إلى علامة العنصر.
        - querySelector(): استرجاع عنصر واحد يطابق المحدد CSS.

        مثال:
        

          let titleElement = document.getElementById("title");
          let paragraphs = document.getElementsByTagName("p");
          let firstElement = document.querySelector(".container");
        


        3. Location Object:
        الكائن Location يحتوي على معلومات حول عنوان URL الحالي ويسمح بالتلاعب به. يحتوي الكائن Location على العديد من الخصائص المفيدة مثل:
        - href: يحتوي على العنوان الكامل للصفحة المحملة.
        - hostname: يحتوي على اسم السيرفر.
        - pathname: يحتوي على المسار النسبي للصفحة.

        مثال:
        

          console.log(location.href);       // الناتج: https://example.com/
          console.log(location.hostname);  // الناتج: example.com
          console.log(location.pathname);  // الناتج: /index.html
        


        هذه هي بعض العناصر الأساسية في Browser Object Model (BOM) واستخداماتها. يتيح لنا BOM تطبيق الوظائف المتعلقة بالمتصفح والعمل مع النوافذ والمستندات والعناصر المتوفرة فيها. تذكر أن واجهة BOM قد تختلف بين المتصفحات، لذا يُفضل أن تتأكد من التوافق مع متصفحات مختلفة عند استخدام أي خاصية أو طريقة معينة.
